# Release Notes
### 1 September 2021
  - Minor updates in the Lab Guide.
  - In [Exercise 4, Task 1](https://github.com/CloudLabs-MCW/MCW-App-modernization/blob/stage/Hands-on%20lab/HOL%20step-by-step%20-%20App%20modernization_08.md), for pushing the code into GitHub, password Authentication has been removed from GitHub.
